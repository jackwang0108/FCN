## 1. Notes:

Please make a soft link to your dataset here:

- for windows

  run the following command in `Administrator`

  ```powershell
  mklink /D <name> <target-path>
  ```

  for example:

  ```powershell
  mklink /D PascalVoc2012-windows pascalvoc2012\VOCdevkit\VOC2012\

- for linux

  ```shell
  ln -s <name> <target-path>
  ```
